# 追剧宝

> 这是一个 **Vibe Coding** 项目 —— 用自然语言对话驱动开发，代码与创意一起生长。

追剧宝是一款基于 Android 的影视聚合播放应用。它不存储视频内容，而是通过用户配置的 API 采集源聚合全网影视资源，提供搜索、播放、续播、投屏、下载等一站式观影体验。

---

## 软件特点

- **Vibe Coding 驱动**：整个项目通过与 AI 的对话式协作逐步迭代而成，从原型到功能不断丰富。
- **多源聚合**：内置多个云端采集站，同时支持用户自定义 API 源，一个应用聚合多平台资源。
- **采集源管理**：可视化的采集源管理页，支持测试、启用/禁用、分类绑定，云端采集站一键添加。
- **智能续播**：自动记录观看进度，再次打开时从上次位置继续播放。
- **播放体验**：基于 DKVideoPlayer，支持多种播放内核切换、手势控制、长按倍速、亮度/音量/进度调节。
- **投屏 & 下载**：支持 DLNA 投屏到电视/盒子，支持视频下载管理。
- **个性化设置**：主题模式、播放内核、解码器、画质偏好、自动画中画等均可自由配置。
- **缓存精细化管理**：独立的缓存管理页，可按类别清理图片、视频、网络缓存、历史记录等，设置项不会被误清。

## 已实现功能

### 首页与内容
- 首页分类展示：电视剧、电影、综艺、动漫、纪录片、体育、短剧等
- 分类浏览与筛选
- 搜索功能（支持搜索历史）
- 视频详情页

### 播放
- 多播放源切换
- 多集数切换与选集
- 自动续播 / 保存播放进度
- 竖屏 / 横屏切换
- 手势操作：单击显隐控制、双击暂停、滑动调节进度/音量/亮度、长按加速
- 播放内核切换：ExoPlayer / IJKPlayer / MediaPlayer
- 画面比例、解码器、画质偏好设置

### 采集源
- 云端采集站列表
- 自定义 API 源添加/编辑/删除
- 采集源测试与分类绑定
- 采集源启用/禁用

### 用户数据
- 观看历史
- 搜索历史
- 本地视频库缓存
- 青少年模式

### 其他
- DLNA 投屏
- 视频下载
- 主题模式（跟随系统/浅色/深色）
- 缓存管理
- 关于与隐私政策

---

## 技术栈

- **UI**：Jetpack Compose + Material Design 3
- **播放器**：DKVideoPlayer（支持 ExoPlayer / IJKPlayer / MediaPlayer）
- **网络**：Retrofit + OkHttp + Gson
- **本地存储**：SQLite + SharedPreferences
- **图片加载**：Coil
- **开发语言**：Kotlin
