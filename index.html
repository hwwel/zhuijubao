核心功能

多源聚合

内置云端采集站，支持自定义 API 源，一个 App 聚合多平台影视资源。

智能续播

自动记录观看进度，下次打开从上次位置继续播放。

强大播放

支持 ExoPlayer / IJKPlayer / MediaPlayer 内核切换，手势控制与长按倍速。

投屏 & 下载

DLNA 投屏到电视/盒子，支持视频下载管理。

缓存管理

按类别清理图片、视频、网络缓存和历史记录，设置项不会被误清。

个性设置

主题模式、画面比例、解码器、画质偏好、自动画中画等自由配置。
