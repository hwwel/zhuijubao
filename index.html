多源聚合

内置多个云端采集站，同时支持自定义 API 源，一个应用聚合多平台资源。

智能续播

自动记录每部影片的观看进度，下次打开直接从上次位置继续播放。

手势播放

双击暂停、滑动调节进度/音量/亮度、长按倍速，操作一气呵成。

一键投屏

支持 DLNA 协议，轻松将视频投屏到电视或盒子，大屏观影更畅快。

下载管理

支持视频下载到本地，WiFi 环境下自动下载，随时随地离线观看。

个性设置

主题模式、播放内核、解码器、画质偏好、自动画中画等自由配置。
